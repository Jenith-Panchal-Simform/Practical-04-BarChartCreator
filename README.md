## Bar chart creator in React

### Features

- Take X and Y axis name from the Form.
- Take the chart data from the form and list them too.
- Depending upon data show the bar chart along with tooltip

---

## How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jenith-Panchal-Simform/Practical-04-BarChartCreator.git
   cd Practical-04-BarChartCreator
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Start Development Server**
   ```bash
   npm run dev
   ```
